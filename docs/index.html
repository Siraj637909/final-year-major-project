🔧 How It Works

Simple Setup, Powerful Results

Get started in minutes with our straightforward setup process

1

Install Dependencies

Clone the repository and install Python packages

git clone https://github.com/Siraj637909/crypto-arbitrage-pro.git pip install -r requirements.txt
2

Configure Settings

Customize exchanges, trading pairs, and alert thresholds

cp config.example.json config.json # Edit with your preferences
3

Start Scanning

Run the scanner and dashboard

python main.py python dashboard.py
4

Monitor & Trade

Watch opportunities appear and execute trades

Open http://localhost:5000 # Get Telegram alerts instantly
